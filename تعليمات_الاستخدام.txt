🚀 خطوات الاستخدام:

1️⃣ فك الضغط عن الملف:
   - انقر بزر الماوس الأيمن على الملف
   - اختر "Extract All" أو "فك الضغط"
   - سيتم إنشاء مجلد باسم "subscriber-management-apk"

2️⃣ تحميل المتطلبات:
   - Java Development Kit (JDK) 11+
     من: https://www.oracle.com/java/technologies/downloads/
   
   - Android Studio
     من: https://developer.android.com/studio

3️⃣ فتح المشروع:
   - افتح Android Studio
   - File → Open
   - اختر مجلد "subscriber-management-apk"
   - انتظر تحميل المشروع (5-10 دقائق)

4️⃣ بناء التطبيق:
   - Build → Build Bundle(s) / APK(s) → Build APK(s)
   - انتظر انتهاء البناء (5 دقائق)

5️⃣ التثبيت على الهاتف:
   - وصل الهاتف بـ USB
   - فعّل "تصحيح USB" على الهاتف
   - اضغط زر التشغيل الأخضر في Android Studio
